# BoatOS koppling

## TFT SPI-display

| Display | Raspberry Pi |
|---|---|
| VCC | 5V |
| GND | GND |
| CS | GPIO8 / CE0 / Pin 24 |
| RESET | GPIO25 / Pin 22 |
| D/C | GPIO24 / Pin 18 |
| SDI | GPIO10 / MOSI / Pin 19 |
| SCK | GPIO11 / SCLK / Pin 23 |
| LED | 3.3V eller 5V |
| SDO | GPIO9 / MISO / Pin 21 |

## MPU6050

| MPU6050 | Raspberry Pi |
|---|---|
| VCC | 3.3V |
| GND | GND |
| SDA | GPIO2 / Pin 3 |
| SCL | GPIO3 / Pin 5 |
